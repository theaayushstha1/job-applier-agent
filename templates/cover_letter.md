# Cover Letter Template

[Your Name]
[Your Email] | [Your Phone] | [Your Location]
[Your LinkedIn] | [Your Website]

[Date]

Hiring Team
[Company Name]
[Location]

Dear Hiring Team,

I'm applying for the [Role] position at [Company]. [Opening sentence about why you're interested - be specific about the company].

[Paragraph about your relevant experience. Reference specific skills from the JD. Mention a project that demonstrates these skills with metrics if possible.]

[Paragraph about why you're excited about THIS company specifically. Reference their mission, product, or recent news.]

I'm ready to [relocate to X / start immediately / contribute to Y]. Thank you for considering my application.

Sincerely,
[Your Name]

## Template Variables

The agent will replace these automatically:

| Variable | Source |
|----------|--------|

| `[Your Email]` | profile.json → personal.email |
| `[Your Phone]` | profile.json → personal.phone |
| `[Your Location]` | profile.json → personal.location |
| `[Your LinkedIn]` | profile.json → links.linkedin |
| `[Your Website]` | profile.json → links.website |
| `[Date]` | Current date |
| `[Company Name]` | Extracted from job posting |
| `[Role]` | Extracted from job posting |
| `[Location]` | Extracted from job posting |

## Tips

- Keep to 250-350 words
- Be specific about the company (research them!)
- Mention a relevant project with metrics
- Match tone to company culture
